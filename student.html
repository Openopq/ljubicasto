Сербская раскладка
Через клавиатуру Gboard
1
Откройте любое приложение с клавиатурой (например, чат в Телеграм)
2
Тапните по полю ввода, чтобы открылась клавиатура, затем нажмите значок ⚙️ настроек над ней
3
Языки → Добавить клавиатуру
4
Выберите Сербский → выберите раскладку (кириллица или латиница) → Готово
Переключение языка: зажмите пробел и проведите в сторону, либо тапните значок 🌐
Системная клавиатура iOS
1
Настройки → Основные → Клавиатура → Клавиатуры
2
Нажмите Добавить клавиатуру…
3
Выберите Сербский (кириллица или латиница — можно добавить сразу обе)
Переключение языка: зажмите значок 🌐 рядом с пробелом, выберите нужный язык
Windows 11
1
Параметры (Win+I) → Время и язык → Язык и регион
2
Добавить язык → найдите Сербский → Далее → Установить
3
Если нужна отдельная раскладка: рядом с языком нажмите ⋯ → Параметры языка → Добавить клавиатуру
Windows 10
1
Параметры → Время и язык → Язык
2
Добавить язык → найдите Сербский → Далее → Установить
3
Откроется список языков → выберите его → Параметры → Добавить клавиатуру
Переключение языка (10 и 11): Win + Пробел или Left Alt + Shift (привычный способ для большинства), либо значок языка в трее
macOS
1
Меню Apple → Системные настройки → Клавиатура
2
Раздел Ввод текста → Изменить
3
Нажмите + внизу слева, найдите Сербский (кириллица/латиница), выберите, Добавить
Переключение языка: значок языка в строке меню, либо Ctrl + Пробел
